# system.sav Offsets

## Offsets of Interest

#### **0x4b7 -** _"Hello Abyss"_ completion flag

`00` = _"Hello Abyss"_ incomplete; _"Deep in Abyss"_ locked.\
`01` = _"Hello Abyss"_ complete; _"Deep in Abyss"_ unlocked
